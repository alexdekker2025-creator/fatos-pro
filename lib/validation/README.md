# Модуль валидации

Модуль для валидации дат рождения и имен в нумерологических расчетах.

## Использование

### Валидация даты рождения

```typescript
import { validateBirthDate } from '@/lib/validation';

const result = validateBirthDate({ day: 15, month: 6, year: 1990 });

if (result.isValid) {
  console.log('Дата корректна');
} else {
  console.error('Ошибка:', result.error);
}
```

### Работа с именами

```typescript
import { sumNameLetters, validateName } from '@/lib/validation';

// Проверка имени
if (validateName('Анна')) {
  // Вычисление суммы букв
  const sum = sumNameLetters('Анна'); // 32
  console.log('Сумма букв имени:', sum);
}
```

## Правила валидации

### Дата рождения

- **День**: 1-31
- **Месяц**: 1-12
- **Год**: 1900 - текущий год
- Дата должна существовать (например, 31 февраля недопустимо)
- Дата не может быть в будущем

### Имя

- Должно содержать хотя бы одну букву русского алфавита
- Пробелы и небуквенные символы игнорируются при подсчете
- Регистр не имеет значения

## Алфавит для расчета

Каждая буква русского алфавита имеет числовое значение:

- А=1, Б=2, В=3, Г=4, Д=5, Е=6, Ё=7, Ж=8, З=9
- И=10, Й=11, К=12, Л=13, М=14, Н=15, О=16, П=17
- Р=18, С=19, Т=20, У=21, Ф=22, Х=23, Ц=24, Ч=25
- Ш=26, Щ=27, Ъ=28, Ы=29, Ь=30, Э=31, Ю=32, Я=33

## Примеры

```typescript
// Корректная дата
validateBirthDate({ day: 15, month: 6, year: 1990 });
// { isValid: true }

// Некорректная дата
validateBirthDate({ day: 31, month: 2, year: 2000 });
// { isValid: false, error: 'Некорректная дата' }

// Будущая дата
validateBirthDate({ day: 1, month: 1, year: 2030 });
// { isValid: false, error: 'Дата рождения не может быть в будущем' }

// Сумма букв имени
sumNameLetters('Анна'); // 32 (А=1, Н=15, Н=15, А=1)
sumNameLetters('Иван'); // 29 (И=10, В=3, А=1, Н=15)
```
