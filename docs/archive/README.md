# Архив документации

Эта папка содержит временные документы и инструкции, которые были созданы в процессе разработки проекта.

## Содержимое

- Инструкции по настройке различных сервисов (Resend, Vercel, мониторинг)
- Руководства по деплою и тестированию
- Сводки по выполненным задачам
- Временные чек-листы и планы

## Примечание

Эти документы сохранены для истории, но больше не используются в активной разработке.
Актуальная документация находится в корневой папке проекта:

- `README.md` - основная документация проекта
- `START_LOCAL.md` - инструкция по локальному запуску (English)
- `КАК_ЗАПУСТИТЬ_ЛОКАЛЬНО.md` - инструкция по локальному запуску (Русский)
