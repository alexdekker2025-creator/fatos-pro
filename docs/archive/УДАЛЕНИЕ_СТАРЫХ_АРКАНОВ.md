# Удаление старых статей арканов

## Проблема
В базе данных сейчас 88 статей арканов (44 старых + 44 новых). Нужно удалить старые статьи, чтобы осталось только 44 новых с полными описаниями.

## Решение

### Шаг 1: Откройте Neon Dashboard
1. Перейдите на https://console.neon.tech/
2. Войдите в свой аккаунт
3. Выберите проект `fatos-pro`
4. Откройте вкладку **SQL Editor**

### Шаг 2: Удалите старые статьи
Скопируйте и выполните этот SQL-запрос:

```sql
-- Удаляем старые статьи арканов (те, у которых короткий контент)
DELETE FROM "Article" 
WHERE category = 'Arcana (Cards)' 
AND LENGTH(content) < 300;

-- Проверяем результат
SELECT COUNT(*) as "Осталось статей" 
FROM "Article" 
WHERE category = 'Arcana (Cards)';

-- Должно быть 44 статьи
```

### Шаг 3: Проверьте результат
После выполнения запроса должно остаться ровно 44 статьи (22 аркана × 2 языка).

### Шаг 4: Проверьте приложение
1. Откройте http://localhost:3001/ru
2. Введите имя и дату рождения
3. Нажмите "Рассчитать"
4. Проверьте, что карты арканов отображаются с полными описаниями (утро, день, вечер, ночь)

## Альтернативный способ (через Node.js)

Если хотите удалить через скрипт:

```javascript
// delete-old-arcana.js
const { PrismaClient } = require('@prisma/client');
const prisma = new PrismaClient();

async function deleteOldArcana() {
  const result = await prisma.article.deleteMany({
    where: {
      category: 'Arcana (Cards)',
      content: {
        // Удаляем статьи с коротким контентом (старые)
        contains: 'Сегодня вы полны энергии'
      }
    }
  });
  
  console.log(`Удалено статей: ${result.count}`);
  
  const remaining = await prisma.article.count({
    where: { category: 'Arcana (Cards)' }
  });
  
  console.log(`Осталось статей: ${remaining}`);
  await prisma.$disconnect();
}

deleteOldArcana();
```

Запустите: `node delete-old-arcana.js`
