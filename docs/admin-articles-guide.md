# Руководство по управлению статьями в админке

## Доступ к админке

1. Войдите на сайт с учетной записью администратора: `admin@fatos.pro`
2. Перейдите по адресу: `/ru/admin` или `/en/admin`
3. Выберите вкладку "Articles" (Статьи)

## Просмотр статей

### Фильтры

В админке доступны три фильтра для поиска статей:

1. **Категория (Category)**:
   - `Destiny Number` - Числа судьбы (1-9, 11, 22, 33)
   - `Destiny Matrix` - Позиции матрицы судьбы
   - `Pythagorean Square` - Ячейки квадрата Пифагора
   - `Arcana (Cards)` - Описания арканов (карт дня)
   - `General` - Общие статьи

2. **Язык (Language)**:
   - `Русский (ru)`
   - `English (en)`

3. **Related Value** - Связанное значение для поиска конкретной статьи

### Информация о статье

Для каждой статьи отображается:
- Заголовок
- Категория (цветная метка)
- Язык (цветная метка)
- Related Value (если указано)
- Содержание (первые 3 строки)
- ID статьи
- Дата последнего обновления

## Редактирование статей

### Как найти и отредактировать статью

1. **Для карт (арканов)**:
   - Выберите категорию: `Arcana (Cards)`
   - Выберите язык: `Русский` или `English`
   - В поле Related Value введите: `arcana_1` (для первого аркана), `arcana_2` (для второго) и т.д.
   - Нажмите на кнопку "Edit" у нужной статьи

2. **Для чисел судьбы**:
   - Выберите категорию: `Destiny Number`
   - В поле Related Value введите число: `1`, `11`, `22` и т.д.

3. **Для матрицы судьбы**:
   - Выберите категорию: `Destiny Matrix`
   - В поле Related Value введите: `matrix_lifePathNumber_5` (для числа жизни = 5)

4. **Для квадрата Пифагора**:
   - Выберите категорию: `Pythagorean Square`
   - В поле Related Value введите: `square_1_3` (цифра 1 встречается 3 раза)

### Форма редактирования

При редактировании статьи доступны поля:

1. **Title** (Заголовок) - название статьи
2. **Category** (Категория) - выберите тип статьи
3. **Language** (Язык) - ru или en
4. **Related Value** (Связанное значение) - формат зависит от категории:
   - Arcana: `arcana_1` до `arcana_22`
   - Destiny Number: `1`, `2`, ..., `9`, `11`, `22`, `33`
   - Destiny Matrix: `matrix_[позиция]_[значение]`
   - Pythagorean Square: `square_[цифра]_[количество]`
5. **Content** (Содержание) - текст статьи

### Сохранение изменений

1. Внесите необходимые изменения
2. Нажмите кнопку "Update" (Обновить)
3. Статья будет сохранена и вы вернетесь к списку статей

## Создание новой статьи

1. Нажмите кнопку "Create Article" (Создать статью)
2. Заполните все обязательные поля (отмечены звездочкой *)
3. Укажите Related Value согласно формату для выбранной категории
4. Нажмите "Create" (Создать)

## Удаление статьи

1. Найдите нужную статью
2. Нажмите кнопку "Delete" (Удалить)
3. Подтвердите удаление

⚠️ **Внимание**: Удаление необратимо!

## Примеры Related Value

### Арканы (Карты)
- `arcana_1` - Маг
- `arcana_2` - Верховная Жрица
- `arcana_22` - Шут

### Числа судьбы
- `1` - Единица
- `11` - Мастер-число 11
- `22` - Мастер-число 22

### Матрица судьбы
- `matrix_lifePathNumber_5` - Число жизни = 5
- `matrix_soulNumber_3` - Число души = 3
- `matrix_dayNumber_15` - Число дня = 15

### Квадрат Пифагора
- `square_1_0` - Цифра 1 отсутствует
- `square_1_3` - Цифра 1 встречается 3 раза
- `square_5_2` - Цифра 5 встречается 2 раза

## Советы

1. **Используйте фильтры** для быстрого поиска нужных статей
2. **Проверяйте Related Value** - от него зависит правильное отображение статей на сайте
3. **Создавайте статьи на обоих языках** (ru и en) для полной локализации
4. **Сохраняйте резервные копии** важных текстов перед редактированием
