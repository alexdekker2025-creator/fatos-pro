# Быстрое обновление статей арканов

## Краткая инструкция

### 1. Заполните новые описания

Откройте файл `prisma/new-arcana-articles-template.sql` и замените все `ЗАПОЛНИТЕ:` и `FILL IN:` на реальные описания арканов.

### 2. Удалите старые статьи (опционально)

В Neon SQL Editor выполните:

```sql
DELETE FROM "Article" WHERE category = 'Arcana (Cards)';
```

### 3. Добавьте новые статьи

Выполните заполненный файл `prisma/new-arcana-articles-template.sql` в Neon SQL Editor.

### 4. Проверьте результат

```sql
SELECT COUNT(*) FROM "Article" WHERE category = 'Arcana (Cards)';
-- Должно быть 46 записей (23 аркана × 2 языка)
```

### 5. Проверьте в приложении

Откройте http://localhost:3001/ru и выполните расчет арканов.

---

## Структура описания (рекомендуемая)

```
Название аркана

Значение: [основное значение]

Символика: [символы на карте]

В раскладах:
- Прямое положение: [интерпретация]
- Перевернутое положение: [интерпретация]

Ключевые слова: [список через запятую]
```
