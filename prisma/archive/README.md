# Архив SQL скриптов

Эта папка содержит временные SQL скрипты, которые использовались для миграций и обновлений базы данных в процессе разработки.

## Содержимое

- Скрипты для вставки данных арканов
- Скрипты для обновления премиум-услуг
- Скрипты для удаления старых данных
- Различные версии seed скриптов

## Примечание

Эти скрипты сохранены для истории. Актуальные миграции находятся в папке `prisma/migrations/`.

Для работы с базой данных используйте:
- `prisma/schema.prisma` - схема базы данных
- `prisma/seed.ts` - основной seed скрипт
- `npx prisma migrate dev` - для создания новых миграций
- `npx prisma db push` - для применения изменений схемы
